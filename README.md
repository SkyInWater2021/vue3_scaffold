## 脚本命令

```shell
pnpm dev # 启动项目
pnpm build # 打包项目文件
pnpm commit  # 提交代码
pnpm lint:prettier # 全局格式化代码
pnpm lint:eslint # 代码格式校验
```

## 项目技术栈

- Vite + TypeScript + Vue3

## 项目规范

- 集成editorconfig配置

- 使用prettier工具

- 使用ESLint检测

- 使用stylelint检测

- git Husky和git commit规范

## 项目便利插件

- 自动导入插件

- import 语句自动整理
